# Примеры файлов для импорта

## Справочник городов (коды Минздрава)

Положите сюда Excel-файл с кодами городов (например, от Минздрава).

**Ожидаемый формат:**

| Колонка A (индекс 0)     | Колонка B (индекс 1) |
|--------------------------|----------------------|
| שם יישוב / Название города | קוד / Код             |
| תל אביב                  | N126                 |
| ירושלים                  | M1187                |

- Первая строка — заголовок (будет пропущена при импорте).
- Колонка с названием города и колонка с кодом задаются в конфиге (`CitiesImportConfig`: `NameCol`, `CodeCol`). По умолчанию: название = 0, код = 1.
- Названия и коды нормализуются (пробелы), дубликаты по названию обрабатываются как «последняя запись побеждает».
- Иврит поддерживается без дополнительной настройки.
